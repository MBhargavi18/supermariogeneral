# 🎮 SUPER MARIO GENERAL QUIZ

## 📦 This Folder Contains:
- `index.html` - Main game file
- `style.css` - Mario theme styling
- `game.js` - Game logic + 300 questions

## 🎯 Questions:
- 100 Math questions
- 100 Science questions  
- 100 Geography questions
- Total: 300 educational questions!

## 🚀 How to Play:
1. Double-click `index.html`
2. Select Mario 🧑 or Barbie 👸
3. Use Arrow Keys to move
4. Press UP or SPACE to jump
5. Hit yellow ? blocks for quizzes!

## 📤 Upload to GitHub:
1. Create repository "supermario-general-quiz"
2. Upload all 3 files
3. Enable GitHub Pages
4. Share: yourusername.github.io/supermario-general-quiz/

Enjoy! 🎮
